Skutils Documentation
=====================

.. image:: _static/skutils_logo.png
   :alt: Skutils Logo


SkuTek Utilities or **skutils** is an open-source python library to control
SkuTek Instrumentation's
`product lines for data acquisition and data management <http://skutek.com/products>`_.
**skutils** runs on your lab computer, but controls your hardware remotely using a REST API.

Installation
------------

**Via Pip**

.. code-block:: shell

   pip install skutils


**From Source**

.. code-block:: shell

   git clone https://gitlab.com/skutek/libraries/skutils.git
   pip install ./skutils

.. Areas of interest:
.. ------------------


.. :doc:`Digitizer Control and Visualization</femtodaqcontrol>`
.. ************************************************************
.. Digitizers are Data Acquisition (DAQ) hardware that digitizes analog signals and performs real-time digitial signal processing. 
.. This includes operations such pulse processing, triggering, and streaming data.

.. SkuTek's Vireo and Kingfisher Digitizers can be controlled from your desktop using the 
.. :any:`FemtoDAQController<skutils.FemtoDAQController.FemtoDAQController>` class. 
.. In the following example, we connect to a Vireo digitizer connected to a computer via 
.. USB and print out some device information.


.. .. code-block:: python
..    :linenos:
   
..    import skutils
..    digitizer = skutils.FemtoDAQController(skutils.FEMTODAQ_USB)
..    print(digitizer.summary())

.. Outputs

.. .. code-block:: text

..    Vireo-000019 (192.168.7.2)
..    Product Revision    : VIREO100_REV_B
..    Number of Channels  : 2
..    Sampling Frequency  : 100.0 MHz
..    ADC Bitdepth        : 14 bits
..    Maximum Wave Length : 81.92us
..    Firmware Version    : 5.5.0-0
..    Software Version    : 5.4.0
..    Linux Image Version : 5.1.1
