Gallery
=======

This gallery is a demonstration of the various ways we can use SkuTils in basic analysis cases.
There is an example with creating a histogram out of realdata taken on a FemtoDAQ Vireo 2-channel digitizer, 
as well as an empty example for a future application note that will have more than one saved data file.
