## SkuTek Launches Open Source Control Library

We are happy to announce the beta release of **SkuTek Utilities**! SkuTek Utilities or **skutils** is an open-source Python library to control SkuTek Instrumentation's product lines for data acquisition and data management. skutils runs on your lab computer, but controls your hardware remotely using a REST API.

Skutils is free, open-source, and pip installable for maximum portability in a lab environment.

Skutils is designed to be used as a part of your Python data analysis pipeline. This allows integration of data collection and analysis together into a single script for automated workflows or easy-to-port experiments. Skutils is also designed for student usage and is ideal for distributing or teaching students how to use Jupyter Notebooks to automate experimental analysis and data collection. Python was chosen due to its prevalence in research and education. We currently have 4 example experiments published with more coming soon :).
